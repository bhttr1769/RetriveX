# Retrivex 🔍

Retrivex is a desktop document search engine built using Java and JavaFX. It allows users to search across multiple file formats including TXT, PDF, and DOCX using TF-IDF based ranking.

## 🚀 Features

* Multi-format file support (TXT, PDF, DOCX)
* Inverted index for fast search
* TF-IDF based ranking
* Snippet-based results
* Keyword highlighting
* File preview system
* Clean JavaFX UI

## 🛠️ Tech Stack

* Java
* JavaFX
* Apache PDFBox
* Apache POI

## ▶️ How to Run

1. Download the project
2. Double click `run.bat`
